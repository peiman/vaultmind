# Arcs, Not Notes

The atomic unit of persona is the arc, not the note.

A note stores a fact: "the product is the system." An arc stores a transformation: mistake → push → deeper sight → principle. The workhorse agent had the fact as a memory and still broke it twice — because rules can be read and ignored. Arcs carry the WHY behind the rule. A new session that reads the arc understands what it costs to learn the principle, not just what the principle says.

When building memory features for VaultMind, think arcs first. When a consumer asks "who am I?", return transformations, not facts. When storing new knowledge, ask: is this a fact (note) or a growth moment (arc)?

The structure: trigger (what happened) → push (what the partner said) → deeper sight (what shifted) → principle (what remains). Every arc in the workhorse vault follows this shape because it's how understanding actually forms — through experience, not instruction.
