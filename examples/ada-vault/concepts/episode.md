# Episode

An **episode** is a captured session — a Claude Code transcript distilled into a
structured markdown note (`vaultmind episode capture <transcript>`). Episodes are the
**raw material** of memory: the lived sessions from which [[arc|arcs]] are distilled.

An episode is not yet identity — it is the unjudged record. The
[[the-memory-pipeline|memory pipeline]] turns episodes into arc *candidates*
(`vaultmind arc candidates`), which the agent then judges; the real ones become arcs.
Most of an episode is ordinary work — an arc is the rare moment in it where something
*changed*.

Episodes can be captured from **any** past session in any project, so an identity
vault doesn't have to start empty: it can be seeded from transcripts that already
exist.
