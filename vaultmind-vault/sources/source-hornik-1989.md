# Hornik, Stinchcombe & White — Universal Approximators (1989)

Hornik, Stinchcombe, and White proved in *Neural Networks* that standard multilayer feedforward networks with a single hidden layer using any squashing activation can approximate any Borel-measurable function on a compact set to arbitrary accuracy, given sufficient hidden units. This is one of the two canonical universal-approximation results (the other being Cybenko 1989) and is the theoretical justification for why a [[multilayer-perceptron|multilayer perceptron]] is a universal function approximator.
