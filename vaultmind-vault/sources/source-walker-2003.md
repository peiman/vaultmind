# Walker et al. — Dissociable Stages of Human Memory Consolidation and Reconsolidation (2003)

Participants learned a motor sequence task across two sessions separated by either sleep or an equivalent waking period. Sleep produced a 20.5% improvement in speed and 39.4% reduction in errors; equivalent waking time produced no benefit. The effect was stage-specific: slow-wave sleep correlated with accuracy gains, while stage-2 NREM (spindle-rich) sleep correlated with speed gains — demonstrating that distinct sleep stages selectively support distinct aspects of procedural memory.

Reconsolidation was also tested: re-exposing consolidated memories to the original learning context (a "reminder" before sleep) produced further sleep-dependent enhancement, while the same reminder before a waking retention period did not help. This established that memory consolidation is not a single-pass process but iterative — each sleep period can further consolidate previously learned material if it is briefly reactivated before sleep.
