## Wikipedia — Microelectrode Array / Utah Array

Reference overview that includes the Utah array: 100 conductive silicon needles arranged in a 10×10 grid on a 4×4 mm substrate, recording from electrode tips only (limiting per-channel information), and the contrast with the Michigan-style planar array's design freedom. Used as the encyclopedic anchor for the Utah array concept note alongside the Maynard 1997 primary source.
