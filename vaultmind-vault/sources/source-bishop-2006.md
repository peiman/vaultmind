# Bishop — Pattern Recognition and Machine Learning (2006)

Christopher Bishop's *Pattern Recognition and Machine Learning* (Springer) is the standard graduate-level textbook for machine learning from a probabilistic perspective. Used here as a reference for [[loss-function|loss functions]] viewed as negative log-likelihoods under explicit probabilistic models, the bias-variance decomposition, and the probabilistic foundations of MLP training.
