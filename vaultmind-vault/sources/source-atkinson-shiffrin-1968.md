# Atkinson & Shiffrin — Human Memory: A Proposed System (1968)

Atkinson and Shiffrin proposed what became known as the Modal Model: a three-stage architecture in which information flows from a large-capacity sensory register (persisting for fractions of a second) into a limited-capacity short-term store (STS, persisting for seconds) and from there into an essentially unlimited long-term store (LTS). The critical transfer mechanism was rehearsal — deliberate recycling of information in the STS — which both maintained items against decay and, with sufficient duration, drove encoding into LTS. The model also specified control processes (coding strategies, attention allocation, retrieval plans) that the individual could deploy to modulate memory performance.

The [[multi-store-model|Multi-Store Model]] dominated memory research for over a decade and provided the first computationally explicit architecture for the entire human memory system. It was eventually challenged by Baddeley and Hitch's (1974) working memory model, which replaced the passive STS with an active, multi-component [[working-memory|Working Memory]] system capable of processing rather than merely holding information. Levels-of-processing theory (Craik & Lockhart, 1972) further challenged the rehearsal-as-transfer assumption, showing that depth of processing mattered more than rehearsal time. Despite these revisions, the three-store framework remains the pedagogical foundation for memory education and the structural template against which alternatives are compared.

VaultMind's architecture is informed by the Atkinson-Shiffrin transfer logic while correcting its over-reliance on rehearsal. The [[Fleeting Note|fleeting note]] → [[Permanent Note|permanent note]] pipeline mirrors the STS→LTS transfer, but the trigger for transfer is elaboration and linking (deep processing, per Craik & Lockhart), not mere repetition. VaultMind's inbox and daily-note workflows implement a timed sensory register: unprocessed captures expire from active surfacing after 48 hours, forcing deliberate processing decisions that parallel the attentional gating Atkinson and Shiffrin described between the sensory register and STS.
