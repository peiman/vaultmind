# Cauchy — Méthode Générale (1847)

Augustin-Louis Cauchy's 1847 note in *Comptes Rendus* introduced what is now called the method of steepest descent for solving non-linear systems — the historical origin of [[gradient-descent|gradient descent]]. The Wikipedia "Gradient descent" history section is used as the canonical secondary reference; the 1847 *Comptes Rendus* note itself does not have a standalone DOI.
