## Wikipedia — Spiking Neural Network

Reference overview of SNNs: neurons that communicate via discrete spikes generated when membrane potential crosses a threshold, the temporal coding-vs-rate-coding distinction, common neuron models (Hodgkin-Huxley, [[leaky-integrate-and-fire|LIF]], [[izhikevich-model|Izhikevich]]), software simulators (Brian, NEST, NEURON), and neuromorphic hardware substrates (SpiNNaker, TrueNorth, Loihi). Notes that SNN accuracy gap with conventional ANNs has shrunk and disappeared on some tasks.
