# Liu et al. — Optogenetic Stimulation of a Hippocampal Engram (2012)

Liu, Ramirez, Pang, Puryear, Govindarajan, Deisseroth, and Tonegawa labelled dentate gyrus neurons that were active during contextual fear conditioning with channelrhodopsin-2, then later, in a different and safe context, optically reactivated those exact cells. Mice froze. The reactivation alone was sufficient to recall the fear memory, and the effect was specific to the cells tagged during learning — random or non-context cells did not produce freezing.

This was the first direct demonstration that an engram (a specific sparse ensemble of neurons) is both necessary and sufficient for memory recall. It moved the engram from a theoretical construct dating back to Richard Semon (1904) and Karl Lashley's failed search for it (1950) into a mechanistically tractable object. The experiment provided proof-of-concept for optogenetic engram manipulation, opening the rest of the Tonegawa programme: false-memory implantation, valence switching, engram silencing in Alzheimer models.

For VaultMind the engram literature reframes "a memory" from a fuzzy associative gradient into a sparse, addressable cell ensemble — closer to the discrete chunk identifiers VaultMind uses. The fact that reactivating the right ensemble retrieves the experience parallels how loading the right note set into context can reconstruct a prior reasoning state.
