# Stickgold — Sleep-Dependent Memory Consolidation (2005)

This *Nature* review established the case for sleep as an active, selective memory processor — not merely a period of reduced interference. Stickgold synthesised the then-emerging evidence for sleep-stage-specific consolidation effects, distinguishing declarative memory (benefiting from SWS), procedural skills (benefiting from late-night REM), and emotional memory (requiring REM-specific processes). The review was one of the first to put forward a unified framework for understanding why different memory types depend on different sleep stages.

Stickgold argued that the sleeping brain is engaged in an active "memory evolution" process — not merely storing traces but transforming them: extracting regularities, generalising across examples, and linking new learning to prior knowledge. This transformation view was prescient; subsequent work on sleep-dependent insight (Wagner et al. 2004) and schema integration (Tamminen et al. 2010) confirmed that sleep does more than stabilise — it reorganises.

For VaultMind, the transformation view of sleep consolidation directly informs the design of background enrichment passes: the offline processing stage should not merely re-index notes but should also infer new links, surface patterns across recently added material, and restructure the knowledge graph based on accumulated context — analogous to the active reorganisation Stickgold describes.
