# Diekelmann & Born — The Memory Function of Sleep (2010)

This is the canonical review of sleep-dependent memory consolidation, cited over 4,000 times. Diekelmann and Born synthesise human and animal evidence for the Active Systems Consolidation (ASC) hypothesis: sleep — specifically NREM slow-wave sleep — serves as a dedicated offline window during which hippocampal memory traces are reactivated (via sharp-wave ripples nested in slow oscillation UP states) and progressively transferred to neocortical long-term stores.

The review provides a comprehensive treatment of the two-stage consolidation model (fast hippocampal encoding, slow neocortical consolidation), the EEG signatures of memory-relevant sleep events (slow oscillations, spindles, SWRs), targeted memory reactivation (TMR) studies, and the emotional memory literature. It carefully distinguishes declarative from procedural memory consolidation, assigning SWS the primary role in declarative and REM a supporting role in procedural and emotional memory.

Born and Diekelmann engage directly with SHY, arguing that the two frameworks are complementary: while SHY describes global synaptic renormalisation (restoring dynamic range), ASC describes the selective overlay of replay-driven potentiation at specific trace synapses. The best-supported synthesis view — which this review articulates — is that both processes co-occur during NREM sleep.
