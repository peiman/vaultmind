# Sweller — Cognitive Load During Problem Solving (1988)

Sweller's 1988 paper established Cognitive Load Theory by demonstrating that conventional problem-solving instruction (means-ends analysis, goal-directed search) imposes heavy working memory demands that interfere with schema acquisition. Experiments compared students learning algebra and geometry using conventional goal-directed problems versus worked examples or goal-free problems. Students using worked examples or goal-free variants consistently showed superior performance on transfer tests, despite (or because of) lower cognitive effort during training.

The core argument: means-ends analysis forces the learner to simultaneously hold the current state, the goal state, all subgoals, and the operators being evaluated — saturating working memory and leaving no capacity for schema construction. Worked examples eliminate the search process, freeing working memory for the germane activity of understanding the solution structure.

This paper launched a research program in instructional design and educational psychology that produced the expertise reversal effect, the split-attention effect, the redundancy effect, and the modality effect — all consequences of the three-load framework (intrinsic, extraneous, germane) that CLT later formalized.
