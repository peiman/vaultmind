# Rosenblatt — The Perceptron (1958)

Frank Rosenblatt's 1958 paper in *Psychological Review* introduced the perceptron as a probabilistic model of how information might be stored and organized in the brain. The paper proposed a learning machine — first simulated on an IBM 704 and later built as the Mark I Perceptron hardware — that adjusted connection weights from labeled examples and was provably able to learn any linearly separable function. This is the foundational reference for the [[perceptron|perceptron]] and one of the earliest formulations of supervised learning by gradient-style updates.
