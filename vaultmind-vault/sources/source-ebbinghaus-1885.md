# Ebbinghaus — Memory (1885)

Ebbinghaus conducted the first systematic experimental study of memory and forgetting using himself as the sole subject and nonsense syllables as material. He discovered that retention drops rapidly in the first hour after learning and then levels off into a slow decline—the shape now universally known as the [[Forgetting Curve]]. He also discovered that distributing practice over time (the [[Spacing Effect]]) dramatically reduced the total study time needed to reach a given retention criterion, a finding that predates any theoretical explanation by decades.

The methodological contribution was as important as the empirical one: Ebbinghaus invented the savings score (relearning time as a measure of residual memory), controlled for interference, and quantified learning curves with mathematical precision. His forgetting curve follows the formula R = e^(−t/S), where S is memory stability—a form still used in modern spaced-repetition systems. The work established memory as a measurable, law-governed phenomenon rather than a mystery accessible only to introspection.

VaultMind's review scheduling system is built directly on Ebbinghaus's two discoveries. The forgetting curve governs when a note is flagged for review: if a note has not been accessed within its predicted retention window, VaultMind surfaces it proactively. The spacing effect governs the interval-growth algorithm: each successful review extends the next interval, matching the exponential spacing logic that maximizes long-term retention for a fixed review budget.
