# Cybenko — Sigmoidal Universal Approximation (1989)

George Cybenko's paper in *Mathematics of Control, Signals and Systems* proved that finite linear combinations of compositions of a sigmoidal function with affine transformations can uniformly approximate any continuous function on the unit hypercube. Together with Hornik et al. (1989), it establishes that a single-hidden-layer [[multilayer-perceptron|MLP]] with a sigmoid activation is a universal function approximator.
