# Tulving & Thomson — Encoding Specificity (1973)

Tulving and Thomson demonstrated that retrieval success depends not just on the stored trace but on the match between encoding context and retrieval cue. A word learned in the presence of one cue word is better recalled with that specific cue than with a word that is a stronger associate in general—even when the stronger associate would normally be a better prompt. This encoding specificity principle unified a fragmented literature on cue-dependent forgetting and context effects under a single framework.

The theoretical implication is that memory is fundamentally relational: what gets stored is not an isolated item but an item-in-context bundle. Retrieval is pattern matching against that bundle, so [[Encoding Specificity]] predicts that the richest retrieval cue is one that reconstructs the original encoding context as closely as possible. This insight was radical in 1973 and remains central to modern theories of episodic memory.

VaultMind applies encoding specificity by capturing rich contextual metadata at note-creation time: active tags, open notes, recent queries, and editing session fingerprints. At retrieval time this context vector is used alongside semantic similarity to re-rank results. A note written during a particular project phase retrieves better when the user is working on that project again, mirroring the context-match advantage Tulving and Thomson measured in the lab.
