# Anderson — The Architecture of Cognition (1983)

Anderson introduced ACT* (predecessor to [[ACT-R]]), a unified theory of human cognition built around two memory systems: declarative memory (facts) and procedural memory (production rules). Declarative memory chunks have an activation level that decays over time and is boosted by use, priming, and associative spread from related chunks. Retrieval succeeds only when a chunk's activation exceeds a threshold, which explains both successful recall and forgetting within one framework.

The book established that memory is not a passive store but a dynamic system shaped by use history and context. Activation decay predicts the forgetting curve; associative priming predicts [[Spreading Activation]] effects; the threshold mechanism predicts tip-of-the-tongue failures and context-dependent recall. These predictions have been validated across decades of experimental psychology, making ACT* and its descendants the most empirically grounded cognitive architectures available.

VaultMind's note-scoring model is directly inspired by ACT-R activation equations. Each vault note carries a computed activation value that incorporates recency (time since last access), frequency (total access count), and associative spread from linked notes. Notes below an activation threshold are deprioritized in retrieval results, mirroring ACT-R's threshold-gated recall and preventing low-relevance material from cluttering responses.
