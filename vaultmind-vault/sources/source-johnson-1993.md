# Johnson, Hashtroudi & Lindsay — Source Monitoring (1993)

This review article established source monitoring as a unified research framework for understanding how people attribute memories to their origins. Johnson and colleagues synthesized a decade of work on reality monitoring (distinguishing perceived from imagined events), external source monitoring (distinguishing between multiple external sources), and internal source monitoring (distinguishing between different internally generated memories). The unifying claim is that all of these are instances of a single inferential process: using the qualitative characteristics of memory records to make attributions about their causal history.

The paper catalogued the features that support accurate source attribution (perceptual vividness, spatial/temporal contextual detail, cognitive operation records) and the conditions that produce errors (source similarity, high imagery ability, divided attention at encoding, delay). It also introduced the distinction between automatic activation (which retrieves content without source) and deliberate recollection (which attempts to recover contextual features) — a distinction with direct implications for understanding when source errors occur.

The source monitoring framework became the dominant account of false memory phenomena in the 1990s and informed subsequent work on eyewitness testimony, suggestion effects, and the DRM paradigm.
