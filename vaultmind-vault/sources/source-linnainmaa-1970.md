# Linnainmaa — Reverse-Mode Automatic Differentiation (1970)

Seppo Linnainmaa's 1970 master's thesis at the University of Helsinki is widely credited as the first explicit description of reverse-mode automatic differentiation — the algorithm that the deep-learning community later rediscovered and popularized as [[backpropagation|backpropagation]]. The thesis was originally framed around analyzing cumulative rounding error in numerical algorithms; Wikipedia's Linnainmaa article is used here as the canonical secondary reference because the original Finnish-language thesis is not online with a stable DOI.
