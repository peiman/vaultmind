# Tulving — Episodic and Semantic Memory (1972)

In this foundational chapter, Tulving introduced the first formal taxonomy of declarative memory by distinguishing [[episodic-memory|Episodic Memory]] from [[semantic-memory|Semantic Memory]]. Episodic memory encodes personally experienced events with their temporal and spatial context — the what, when, and where of lived experience. Semantic memory, by contrast, stores general world knowledge: facts, concepts, and word meanings stripped of the autobiographical context in which they were originally acquired. Tulving argued these two systems differ not only in content but in the nature of their input and retrieval operations, with episodic memory being more vulnerable to forgetting and contextual interference.

The distinction was theoretical at the time of writing — Tulving drew primarily on behavioral evidence and logical argument rather than neuroimaging — but it proved extraordinarily generative. Subsequent decades of neuropsychological and neuroimaging research supported the dissociation: patients with hippocampal damage can lose episodic memory while preserving much of semantic memory, and fMRI studies show partially distinct activation patterns for the two systems. The episodic/semantic taxonomy became the dominant framework for understanding declarative memory and remains the foundation of virtually every memory model in cognitive science and AI.

VaultMind's note architecture maps directly onto Tulving's distinction. [[Concept]] notes and [[MOC|Maps of Content]] function as semantic memory — stable, decontextualized knowledge about a domain. [[Fleeting Note|Fleeting notes]], [[Daily Note|daily notes]], and session logs function as episodic memory — timestamped records of what the user was thinking, reading, and working on at a specific moment. This architectural separation allows VaultMind to retrieve by concept (semantic) or by context (episodic) depending on the user's query type, echoing the dual-system structure Tulving first articulated.
