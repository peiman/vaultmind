# Plihal & Born — Effects of Early and Late Nocturnal Sleep on Declarative and Procedural Memory (1997)

This study provided the first clean within-subjects dissociation of sleep-stage contributions to memory consolidation, and remains one of the most influential sleep and memory experiments. Plihal and Born used a half-night design: subjects slept either the first 3 hours of the night (SWS-rich) or the last 3 hours (REM-rich) after learning. This elegantly disentangled the effects of sleep stage from those of time of day.

The key findings were doubly dissociable. Early sleep (SWS-rich) preferentially improved retention of declarative materials (word pairs, spatial location memory) relative to wakefulness, while producing no advantage for procedural mirror-tracing. Late sleep (REM-rich) showed the opposite: preferential improvement in procedural mirror-tracing and perceptual skills, with minimal benefit for declarative material. The double dissociation across memory types and sleep stages provides the strongest possible evidence that the two stages serve different memory functions.

These results framed all subsequent sleep-and-memory research. They established that asking "does sleep consolidate memory?" is underdetermined — the relevant question is which stage consolidates which type of memory. Declarative memory needs SWS; procedural memory benefits from REM. This stage-specificity implies distinct mechanisms: hippocampal replay during SWS for declarative; neocortical replay during REM theta for procedural.

For VaultMind, the double dissociation is a design constraint: different types of knowledge structures (factual notes, procedural how-to guides, relational links) may benefit from different consolidation treatments in the offline pass — not a single homogeneous maintenance operation.
