## About

Endel Tulving was an Estonian-Canadian cognitive psychologist best known for distinguishing episodic memory (personal events) from semantic memory (general knowledge) and for formulating the principle of [[encoding-specificity|Encoding Specificity]]. His 1973 paper with Thomson established that retrieval success depends critically on the match between encoding context and retrieval cues.

## Key Contributions

Tulving's encoding specificity principle has a direct design implication for VaultMind: notes retrieved at query time are only as useful as their contextual overlap with the active working context. This motivates storing rich contextual metadata at note-creation time — tags, related IDs, and session context — so that recall cues can find a meaningful match. His episodic/semantic distinction also informs VaultMind's note taxonomy: source notes capture episodic records (when something was read or decided), while concept notes encode semantic knowledge that persists across sessions.
