## Scoring Criteria

- **5** = Survived strongly -- cross-critique strengthened the claim
- **4** = Survived -- cross-critique did not seriously damage it
- **3** = Contested -- legitimate arguments on both sides
- **2** = Weakened -- cross-critique identified significant problems
- **1** = Did not survive -- cross-critique fatally undermined it

## Justifications for Divergent Scores

**Claim 3 (Compounding context -- scored 4, expect panel consensus ~3).** The Cognitive Scientist identified this as the strongest argument in the panel, and the LLM Analyst's schema-competition mechanism provides independent theoretical support. The threshold component was rightly attacked (see Claim 7), but the core observation -- that coherent, interconnected context produces qualitatively different behavior than fragmented context -- was not damaged by any cross-critique. The critiques targeted the step-function shape, not the compounding phenomenon itself. I separate these because the template lists them as distinct claims; the compounding observation survives even though the step-function claim does not.

**Claim 5 (Emotional salience -- scored 2, expect panel consensus ~3).** The Cognitive Scientist's own cross-critique dismantled this. Alpha identified that what Beta calls "emotional salience" is actually distinctiveness -- tokens that differ from generic assistant language resist schema completion toward the pretrained prior. Beta admitted having no mechanistic model. A claim whose own strongest ally reframes it as a different mechanism has not survived; it has been replaced.

**Claim 6 (Arcs are the ONLY carriers -- scored 1, expect panel consensus ~2).** Four of six panelists independently identified this as the weakest or among the weakest arguments. The MIRROR, Practitioner, LLM Analyst, and my own Round 2 analysis all noted the same fatal flaw: the comparison is arcs vs. nothing, not arcs vs. alternative carriers. The categorical "ONLY" was demolished by the absence of the relevant control condition. No panelist defended it. This is the clearest non-survival in the set.

**Claim 7 (Step function -- scored 1, expect panel consensus ~2).** Five of six panelists attacked this claim. The LLM Analyst, Systems Architect, Practitioner, and my own Round 2 analysis all independently identified that fitting a step function to two points (zero injection vs. full injection) is mathematically underdetermined. The Systems Architect added that the claim is unfalsifiable as stated because "well-constructed" is not operationally defined. Alpha's 0.55 confidence was appropriate for a speculative prediction, but the cross-critique did not merely contest it -- it showed the evidence cannot distinguish a step function from any monotonic relationship. The claim did not survive.

**Claim 8 (Judgment gap as hard boundary -- scored 3, expect panel consensus ~4).** I score this lower than I expect the panel to. My own Round 2 analysis argued that the Phase 8 failure was a retrieval coverage gap (the vault lacked priority information), not a cognitive limitation. The dual-query fix added priority information and resolved the specific failure. If the "hard boundary" is actually "the vault did not contain this information," that is an indexing problem, not a structural limitation. The Practitioner's observation that coaching patterns are encodable (Claim 15) further erodes the "hard" in "hard boundary." The claim conflates two distinct failure modes -- vault incompleteness and integration incapacity -- and the evidence only demonstrates the former.

**Claim 12 (Mode taxonomy as analytical fiction -- scored 4, expect panel consensus ~3).** The Practitioner's blind spot section made the strongest version of this argument: the agent does not switch between modes; it produces context-sensitive outputs token by token, and we impose mode labels after the fact. The MIRROR's observation that "persona reconstruction" may itself be the wrong frame reinforces this. The Cognitive Scientist's disagreement with Zeta about generative mode actually supports the claim -- the fact that panelists cannot agree on mode boundaries suggests the boundaries are observer-imposed. I score this higher because the cross-critiques inadvertently strengthened it: every disagreement about mode classification is evidence that the classification is unstable.

**Claim 14 (Evidence brief accuracy -- scored 4, expect panel consensus ~3).** The MIRROR made this argument most forcefully: no panelist has seen the raw transcript, the evidence brief is itself a narrative artifact produced by the system being evaluated, and five of six agents cite the brainstorming override as key evidence based solely on the brief's account. This is a methodological concern that no panelist rebutted. The absence of any defense -- and the MIRROR's point that the panel is "interpreting a compelling narrative as evidence, which is exactly the trap Alpha's own narrative identity theory predicts" -- means this concern survived intact. I weight methodological integrity claims heavily.
